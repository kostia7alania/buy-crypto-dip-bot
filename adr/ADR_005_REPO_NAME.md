# ADR 005 — Repository name

## Decision

Use `buy-crypto-dip-bot`.

## Context

The repo is intentionally crypto-focused and keyword-heavy. It should answer the search intent directly.

## Alternatives

- `buy-the-dip-crypto-bot`
- `crypto-dip-bot`
- `crypto-dca-guard`
- `dca-guard`

## Consequences

- Clear crypto focus.
- Clear bot focus.
- Product name remains cleaner: `DCA Guard`.
