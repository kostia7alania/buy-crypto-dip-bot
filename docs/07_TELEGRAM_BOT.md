# Telegram Bot

## MVP commands

```txt
/start
/status
/risk
/strategies
/orders
/audit
/help
```

## Later commands

```txt
/pause
/resume
```

## Notifications

```txt
🟡 Signal detected
🔴 Risk rejected signal
🟢 Dry-run order created
⚠️ Live mode attempt rejected
🚨 Market data error
```

## Forbidden in MVP

- Buy Now button
- live trading button
- withdrawal commands
- leverage/futures commands

## Bot framework

grammY.
