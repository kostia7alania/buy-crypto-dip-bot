# Product Spec

## Product name

DCA Guard

## Repository

buy-crypto-dip-bot

## One-liner

AI Crypto DCA Bot for Buying the Dip with dry-run mode, strict risk limits, Telegram alerts, and auditable decisions.

## Target users

- crypto investors
- BTC spot buyers
- users who want dip-buying automation
- users who need risk limits before order execution
- builders testing trading automation safely

## Jobs to be done

1. Watch BTCUSDT market data.
2. Detect meaningful dips.
3. Reject unsafe signals.
4. Create dry-run orders first.
5. Alert user via Telegram.
6. Show risk-first dashboard.
7. Provide SEO landing pages for acquisition.

## MVP

- Bybit public market data
- BTCUSDT
- Dip rules
- RiskGuard
- Dry-run ledger
- Audit logs
- Telegram status/alerts
- Nuxt SEO pages
- Dashboard shell

## Non-goals

- No real trading in Task 0.
- No futures.
- No leverage.
- No martingale.
- No meme coins.
- No withdrawals.
- No portfolio optimizer.
- No multi-exchange in MVP.
